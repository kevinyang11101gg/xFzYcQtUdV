# django智慧医疗系统

## 核心功能
- 用户注册与登录
- 管理人员界面
- 前端采用bootstrap框架
- 后端基于django
- 数据库使用mysql

## 适用场景
- 医疗机构信息管理
- 医疗数据分析
- 医疗资源调度

## 技术特点
- 前后端分离，易于维护和扩展
- 基于Python的Django框架，开发效率高
- 使用MySQL数据库，数据存储安全可靠
